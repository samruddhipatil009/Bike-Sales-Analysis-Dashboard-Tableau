# Bike-Sales-Analysis-Dashboard-Tableau

Interactive Tableau dashboard analyzing bike store sales 2011-2016: revenue trends, top products, and customer demographics.

### Dataset
- **File**: `bike_sales_raw_data_2011_2016.csv`
- **Source**: Udemy Course - 'Data Analysis | SQL, Tableau, Power BI & Excel' by Graeme Gordon
- **Size**: ~10,000+ rows
- **Fields**: Order Date, Customer Age, Product Category, Revenue, Country, etc.

### Tools Used
Tableau Public, Excel, Data Cleaning, Data Visualization

### Key Insights
1. Peak sales in age group 35-54
2. Mountain Bikes generated highest revenue
3. Q4 2015 had 23% YoY growth

### Files in Repo
- `data/bike_sales_raw_data_2011_2016.csv` - Raw dataset used for analysis
- `dashboard/Bike_Sales_Analysis.twbx` - Tableau packaged workbook 
- `screenshots/dashboard_overview.png` - Main dashboard preview
- `README.md` - Project documentation
